# ESP32-C3 固件烧录指南

## 📦 已下载的固件文件

您的固件已成功下载至 `downloaded_firmware` 目录，包含以下文件：

### 1. 单独的二进制文件（位于 `firmware-bin/`）
- **bootloader.bin** (20.2 KB) - 启动加载器
- **partition-table.bin** (3 KB) - 分区表
- **high_frequency.bin** (858.19 KB) - 应用程序

### 2. 打包文件（位于 `firmware-flash-package/`）
- **firmware-flash.zip** (542.21 KB) - 上述三个文件的压缩包

---

## 🔧 烧录方法

### 方法一：使用 esptool.py（推荐）

```bash
# 进入固件目录
cd downloaded_firmware/firmware-bin

# 烧录到ESP32-C3（将COM3替换为您的实际端口号）
esptool.py --chip esp32c3 --port COM3 write_flash \
  0x0 bootloader/bootloader.bin \
  0x8000 partition_table/partition-table.bin \
  0x10000 high_frequency.bin
```

### 方法二：使用 ESP-IDF 工具

```bash
# 如果已安装ESP-IDF环境
idf.py -p COM3 flash monitor
```

### 方法三：使用 Flash Download Tools（Windows图形界面）

1. 下载 Espressif Flash Download Tools
2. 选择 ESP32C3 芯片
3. 添加三个bin文件并设置地址：
   - bootloader.bin → 0x0
   - partition-table.bin → 0x8000
   - high_frequency.bin → 0x10000
4. 点击 START 开始烧录

## 📋 引脚配置

| GPIO | 功能 | 说明 |
|------|------|------|
| GPIO3 | PWM输出 | 高频PWM信号（10kHz-300kHz） |
| GPIO4 | LED状态灯 | 低电平点亮 |
| GPIO9 | 按键输入 | 短按切换PWM，长按5秒恢复出厂 |
| GPIO18 | USB D- | USB数据负（专用） |
| GPIO19 | USB D+ | USB数据正（专用） |

## 🔍 USB CDC 调试日志

烧录完成后：

1. 通过USB连接ESP32-C3到电脑
2. 在设备管理器中找到新的COM端口（ESP32-C3 USB CDC）
3. 使用串口工具打开该端口（波特率任意，因为USB虚拟串口不需要固定波特率）
4. 您将看到类似以下的启动日志：

```
I (xxx) MAIN: USB CDC initializing...
I (xxx) MAIN: USB CDC initialized successfully
I (xxx) MAIN: System starting with USB CDC console
```

## ⚙️ 功能说明

### WiFi配网
- 使用微信小程序"乐鑫AirKiss"进行配网
- 确保手机和ESP32-C3在同一2.4GHz WiFi网络

### MQTT控制
订阅以下主题控制设备：
- `harmony/pwm/switch` - 发送 "ON"/"OFF" 控制PWM开关
- `harmony/pwm/freq` - 发送频率值（10000-300000）
- `harmony/pwm/duty` - 发送占空比（0-255）

设备会发布状态到：
- `harmony/pwm/state` - 格式：`freq:xxx,duty:xxx,en:x`

### 恢复出厂设置
两种方式：
1. 长按GPIO9按键5秒以上
2. 15秒内连续上电10次

## ❓ 常见问题

### Q: 找不到COM端口？
A: 
- 检查USB线是否支持数据传输
- 安装CH340或CP2102驱动（如果需要）
- Windows 10/11通常自动识别

### Q: 烧录失败？
A:
- 确认端口号正确
- 尝试按住BOOT键后上电进入下载模式
- 检查波特率设置（默认115200）

### Q: 没有日志输出？
A:
- 确认使用了正确的COM端口（ESP32-C3 USB CDC，不是UART）
- 检查串口工具设置
